# Codebase-exploration block

Paste the block below into your root `CLAUDE.md` / `AGENTS.md`. The tools
named are an example stack — swap in your stack's equivalents; the
discipline matters, not the specific tools.

```markdown
## Codebase exploration

- Outline before content: structural queries (search, symbols, file
  outline) before whole-file reads; open a full file only when you need
  literal content — usually because you're about to edit it
- Native Read of a file before editing it
- Optional: dedicated structural-query tools (e.g. grepika, tilth,
  cachebro) cut context consumption further — swap in your stack's
  equivalents; the discipline matters, not the specific tools
```
